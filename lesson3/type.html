Heading Font

For the heading font, I chose to import "Itim" from Google Fonts. It's an interesting, fun looking font, yet still very easy to read.

Secondary Font

For the secondary font, I chose to import "Noto Sans" from Google Fonts. It is a very legible sans-serif font, and it still has character.
